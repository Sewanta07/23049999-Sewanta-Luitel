Movie Ticket Booking Management System - Setup Guide

1. Open Oracle SQL Developer.
2. Open and run Database/cinema_database.sql.
3. Ensure your Oracle database service is running.
4. Open this project in Visual Studio.
5. Verify the connection string in Web.config (MovieTicketDb).
6. Run the application.

Notes:
- This project uses Oracle.ManagedDataAccess.
- The SQL script creates USER_TABLE, MOVIE, THEATER, HALL, SHOWS, BOOKING, and TICKET.
- Sample data is included in the script for quick testing.
